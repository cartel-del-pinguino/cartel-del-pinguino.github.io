Introducción

Bienvenidos al curso de Linux "El Cártel del Pingüino", un viaje completo para dominar el mundo del software libre y la administración de sistemas con Linux. Este curso está diseñado para quienes desean aprender algo nuevo y no saben como iniciar hackear. El propósito de este curso es darte ese impulso para hacer un cambio y que no te veas abrumado de aprender nuevas cosas, nosotros como estudiantes sabemos lo que es lanzarse a algo nuevo sin tener alguna idea, por lo cual queremos que se pierda el miedo de dar ese salto y que siempre intentes hacer cosas nuevas.

Temario

  • 1. Introducción a Linux
    • - ¿Qué es Linux?
    • - Distribuciones populares
    • - Software libre y licencias
  • 2. Instalación de Linux
    • - Elección de la distribución adecuada
    • - Proceso de instalación
    • - Configuración inicial del sistema
  • 3. Uso del Terminal
    • - Comandos básicos de navegación
    • - Gestión de archivos y permisos
    • - Administración de procesos
    • - Gestión de paquetes y repositorios
  • 4. Control de versiones con Git y GitHub
    • - Conceptos de control de versiones
    • - Comandos básicos de Git
    • - Flujos de trabajo y colaboración en GitHub
  • 5. Entornos Gráficos
    • - Principales entornos de escritorio (Gnome, KDE Plasma, etc.)
    • - Configuración y personalización
    • - Gestores de ventanas ligeros (i3, bspwm, Hyprland)
  • 6. Editores de Texto en Linux
    • - Emacs: Introducción y comandos esenciales
    • - Vim/Nvim: Navegación y edición avanzada
    • - Nano: Edición rápida y fácil
