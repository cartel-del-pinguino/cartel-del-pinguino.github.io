El Cártel del Pingüino

Curso de Linux - Aprende a dominar el sistema operativo del pingüino

Introducción

Bienvenidos al curso de Linux "El Cártel del Pingüino", un viaje completo para dominar el mundo del software libre y la administración de sistemas con Linux. Este curso está diseñado para quienes desean aprender desde cero hasta un nivel avanzado, explorando comandos, configuraciones y herramientas esenciales del entorno Linux. Únete a nuestra comunidad y aprende a manejar el sistema operativo que impulsa servidores, supercomputadoras y millones de dispositivos alrededor del mundo.

Temario

  • 1. Introducción a Linux
    • ¿Qué es Linux?
    • Distribuciones populares
    • Software libre y licencias
  • 2. Instalación de Linux
    • Elección de la distribución adecuada
    • Proceso de instalación
    • Configuración inicial del sistema
  • 3. Uso del Terminal
    • Comandos básicos de navegación
    • Gestión de archivos y permisos
    • Administración de procesos
    • Gestión de paquetes y repositorios
  • 4. Control de versiones con Git y GitHub
    • Conceptos de control de versiones
    • Comandos básicos de Git
    • Flujos de trabajo y colaboración en GitHub
  • 5. Entornos Gráficos
    • Principales entornos de escritorio (Gnome, KDE Plasma, etc.)
    • Configuración y personalización
    • Gestores de ventanas ligeros (i3, bspwm, Hyprland)
  • 6. Editores de Texto en Linux
    • Emacs: Introducción y comandos esenciales
    • Vim/Nvim: Navegación y edición avanzada
    • Nano: Edición rápida y fácil

Metodología

El curso combina teoría con prácticas en tiempo real. Cada módulo incluye ejercicios interactivos y tareas para aplicar lo aprendido en escenarios reales. Al finalizar, serás capaz de administrar sistemas Linux, trabajar con control de versiones, y personalizar entornos de trabajo según tus necesidades.
